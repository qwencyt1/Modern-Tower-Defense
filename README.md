# Modern-Tower-Defense
A python game in the style of Tower Defense. Суть игры не дать дронам захватчикам дойти до базы игрока. У игрока в наличии 3 башни(2 атакующие, 1 башня подержки). Башня подержка - дрон котоорый разносит боеприспасы 2 атакующим башням. Есть 5 видов врагов: Обычный(среднее количество хп и скорость), быстрый(большая скорость но мало хп), щитовик(хп как у средняго но есть щит), камикадзе(такойже как и быстрый но если дойдет до базы игрока то нанесет 3 урона) и босс(медленный, много хп и спавнится каждые 5 волн). Хп всех противников увеличивается каждые 2 волны.
